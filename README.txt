Aquí se encuentra el código soportado por Arduino UNO y NANO (al menos)
y que nos permite hacer girar un motor brushless.

En las primeras líneas se puede elegir en que pin conectar el motor, 
por defecto pone el 8, pero debemos elegir uno PWM para que se pueda 
mandar una señal analógica.

Este código esta enlazado desde www.dronsrj.blogspot.com
Podeis utilizarlo para actividades no lucrativas sin ningún permiso.
